# 🌟 Featured Projects

## 🧠 COGXAR – Hardware Neural Inference Engine

Real-time elderly fall detection system built on the Cypress PSoC 5LP.

### Highlights

- Hardware accelerator designed in Verilog
- ARM Cortex-M3 + Hardware Co-processing
- Sensor fusion using:
  - MPU6050
  - BMP280
  - KY-038
- 2.27× speedup over software implementation
- IEEE manuscript under preparation

---

## ⚛️ Fault-Tolerant Quantum Dot Cellular Automata Multiplexer

Designed four iterations of a nanoscale 2:1 QCA multiplexer.

Achievements:

- Fault tolerance improved:
  - 37.4% → 89.11%
- Lower energy dissipation
- Reduced cell count
- Better than existing published design
- IEEE manuscript under preparation

## 📡 MXene Flexible Patch Antenna

Designed a 2×2 microstrip patch antenna array for the 5.8 GHz ISM band.

Results

- Return Loss: **−26.07 dB**
- VSWR: **1.14**
- Designed in ANSYS HFSS
- FR4 substrate with MXene conductor
